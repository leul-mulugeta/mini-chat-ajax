# 💬 Mini-Chat-Ajax

> 🚧 **Projet en cours de développement**

## 🛠️ Présentation
**Mini-Chat-Ajax** est une application de messagerie instantanée légère. Elle permet aux utilisateurs d'échanger des messages en temps réel via une interface web, en utilisant les technologies **Ajax** et la librairie **jQuery** pour interagir avec un serveur Apache (PHP/MySQL).

## 💻 Stack Technique
- **Frontend** : **HTML5**, **CSS3**, **JavaScript**
- **Librairie** : **jQuery**
- **Backend** : **PHP** (Serveur Apache)
- **Base de données** : **MySQL**

## ⚙️ Installation et Exécution

1.  **Cloner le dépôt** :
    ```sh
    git clone https://github.com/leul-mulugeta/mini-chat-ajax.git
    cd mini-chat-ajax
    ```

2.  **Lancer l'application** :
    - Ouvrez simplement le fichier `index.html` dans votre navigateur web.

## 📁 Structure du projet
```
mini-chat-ajax/
├── css/
│   └── style.css       # Styles de l'application
├── index.html          # Interface principale (client web)
├── script.js           # Logique JavaScript (Ajax / jQuery)
```
